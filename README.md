# UI-Labs 

UI Labs is a component library that is developed to import components for the projects.
- Easily customizable
- Highly scalable

## Application Links
[UI Labs Live](https://ui-labs-sr.netlify.app)

## Components
- Alert
- Avatar
- Badge
- Button
- Cards
- Form
- List
- Modal
- Navbar
- Responsive Grid
- Responsie image
- Typography
